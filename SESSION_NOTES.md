# Session Notes

## Current State
- App purpose: local/cross-platform Illumio blocked traffic extractor with embedded web UI.
- Main workflow: query blocked traffic in 1-day chunks across a selectable number of days, deduplicate flows, export CSV, and provide post-fetch analytics.
- Hosting: server host and port are configurable via `-host`, `-port`, and `-open-browser`, plus `ITT_HOST`, `ITT_PORT`, and `ITT_OPEN_BROWSER`.

## Features Added This Session
- Fixed cancellation deadlock and improved run-finished state handling.
- Improved discovery error handling.
- Hardened traffic JSON parsing and 429 cooldown behavior.
- Added configurable `Days To Fetch` instead of always using 90 days.
- Added post-fetch analytics page at `/summary`.
- Added:
  - Port / protocol summary
  - Environment cross-talk matrix
  - App-to-app matrix
  - Top talkers
  - Unmanaged / external summary
  - Heatmap-style environment and app matrices
- Heatmaps now exclude `External/Unmanaged` by default.
- Labeled unmanaged endpoints are grouped by their labels instead of being forced into `External/Unmanaged`.
- Added CSV import on the analytics page so old exports can be reloaded after restart.
- Added show/hide toggles for API key and secret fields.
- Added protocol mapping for `2 -> IGMP`.
