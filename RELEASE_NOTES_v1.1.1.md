# Illumio Blocked Traffic Extractor v1.1.1

## Highlights
- Hardened API request handling without changing normal in-app browser use
- Reduced local credential exposure risk for saved PCE profiles
- Added method and same-origin enforcement for state-changing routes

## Included In This Release

### Security Hardening
- Changed saved profile storage so `pce_profiles.json` is written with owner-only permissions on supported systems
- Enforced expected HTTP methods across the embedded web server routes
- Added same-origin checks for state-changing browser requests to reduce CSRF-style misuse
- Preserved normal browser behavior for users interacting through the app itself

### Validation
- Verified with `go test ./...`
- Verified with `./smoke_test.sh`
- Rebuilt binaries for Linux, Windows, macOS Intel, and macOS Apple Silicon

## Build Artifacts
- `IllumioTrafficTool_Linux`
- `IllumioTrafficTool_Windows.exe`
- `IllumioTrafficTool_MacOS_Intel`
- `IllumioTrafficTool_MacOS_AppleSilicon`

## Known Caveats
- Firefox may still prompt to save credentials because of browser heuristics
- CSV import expects CSVs generated by this tool's schema
- Live smoke tests require valid PCE credentials and network reachability
- The app still has no built-in login/authentication layer for hosted multi-user exposure
